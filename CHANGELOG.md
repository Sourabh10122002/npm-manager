# Changelog

All notable changes to the "npm-manager" extension will be documented in this file.

## [0.1.0] - 2026-03-28

### ✨ Initial Release

- **Dashboard:** Modern, visual NPM package manager dashboard.
- **Search:** Live NPM registry explorer.
- **Management:** Install, Uninstall, and Update dependency support.
- **Performance:** Native HTML/CSS/JS architecture for lightning-fast performance.
- **Branding:** Professional "npm" red packaging icon.
